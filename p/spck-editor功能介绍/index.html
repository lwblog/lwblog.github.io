Spck Editor功能介绍

之前介绍过一款手机编程的app，那就是Spck Editor，如果没看过的可以去找一找我之前的文章，本篇文章主要讲解Spck Editor的一些设置，还有一些实用的功能，技巧等等。

教学开始

设置

我们打开Spck Editor的设置页面后，大概会看到这些设置

  • GIT
  • 外观
  • 代码宝
  • 编辑器
  • 预览
  • 移动
  • 触控功能

关于git功能，我们留到后面再说。 首先说说外观的设置。

外观

第一，动画效果。其实也就是打开菜单的那一瞬间的动画，开不开也无所谓。

第二，语言。这个就不用多说了吧？就只是选择语言而已。

第三，主题。其实也就是语法高亮，根据你使用的编程语言显示不同的颜色增强可读性。这里我以Python为例子，展示所以主题的语法高亮，选择一个你喜欢的就好。

{ width=50% }

第四，Amoled。编辑器背景颜色，如果没打开的话你的Spck Editor背景会是黑灰色的，打开后变成纯黑。另外打开后会节省电池(反正说明是这样写的，能不能省咱也不知道)

第五，顶栏透明度。顾名思义，可以调节顶部栏的透明度，小白我个人是比较喜欢纯黑(Solid)，而transparent就是透明的。

代码宝

没什么用，无视。

编辑器

第一，自动补全/语法检查。打开后会自动识别你输入的代码，并且帮你补全。而语法检查仅使用于HTML除外的所有编程语言，系统会检测你的代码，并找出错误的语法，以下为示意图，建议打开。
