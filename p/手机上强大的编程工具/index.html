一部手机也能用来编程？

大家好，我是小白，一个想用手机创造奇迹的人。不知道大家有没有想过，有时出门不方便带着电脑，但又手痒想敲几段代码，却发现电脑不在身边。这个时候要是手机也能敲代码就好了…当然，这篇教学文章也是送给那些想学编程，奈何没有条件用电脑的人，让他们用手机实现梦想。

开始教学

经过一番寻找，小白发现了一个编程app，那就是"SPCK Editor"。这是目前小白认为手机上最强大的编程工具了，也是小白现在一直用着的。

首先介绍一下这款app。Spck Editor是一款实用性很强的编程工具，而该软件内更是支持11种语言包括中文（繁体简体都有），所以根本不需要担心看不明白怎么使用。

另外，经过小白实测，以下为spck editor支持的编程语言。

  • HTML, JavaScript, CSS
  • PHP
  • C#, C++
  • Java
  • Python
  • React
  • Vue
  • TypeScript
  • SQL

总共12种编程语言，大家可以看看小白的截图，另外还可以用来编辑txt, md文件。

以上都是小白测试后得到的结果，或许还有哪些被遗漏了，大家可以自行测试。

其实Spck Editor还有好多强大且实用的功能，这里就留到下篇文章在详细的说吧，本文章主要先教学如何下载，和下载后的基础操作。

下载教学

首先，点击此处找到spck editor的zip文件并下载到手机，进行解压后，你就会看到一个apk文件，安装即可。 除了安装app，你也可以直接前往spck的在线版本，当然小白我还是比较推荐下载的。大家可以按照自己的喜好决定。

下载后，打开spck editor会看到这样的一个界面。

这些都是官方自带的项目，要是感兴趣可以点开看看源码，但我们今天的主题不在这里。

考虑到有些人下载到的可能是英语版，所以先说说怎么切换语言。

切换语言

首先点击左上角打开菜单，点击最右边的设定。

找到language并点击打开选项，往下滑找到中文即可。

创建项目

语言也切换完成了，接下来我们来创建一个项目。一样点开左上角，点击"+“的按钮，选择新建项目。这里会看到各种项目的模板，其实就是自动生成空的文件，例如点击HTML模板，就会自动帮你生成空的html, css和js文件。不想要自动生成的话选择"No Framework"就行。

最后，给项目命名再点击创建即可。这样就创建了一个新的项目了。接下来测试一下。敲完代码点击右上角绿色的按钮就可以运行了。又或者你可以用浏览器输入localhost:7700查看运行结果。

另外，其实spck的功能还有很多很多，关于其他功能的详细介绍，小白会放在其他的文章。
