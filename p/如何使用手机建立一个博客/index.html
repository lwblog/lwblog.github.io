如何在手机使用hugo建立一个个人博客

前言

最近心血来潮，想创建一个教学类的博客，但又没（懒）时（得）间一点一点的用代码敲出来，难道就没有方法直接生成一个博客出来吗？
找着找着发现了hugo这么个东西，我一看，哎，好像还不错，那就用hugo来建一个博客吧。于是乎经过了一段查询教学的时间发现为什么网上的教程都是使用电脑（废话），难道是看不起手机？（并没有）
于是爱搞事情的我有了个大胆的想法，那就是用手机通过hugo搭建一个博客！想要通过手机使用hugo创建，最大的问题就在于终端，那么有什么办法解决呢？那就是手机神器，“termux”！
那么废话不多说了，开始教程！（虽然已经很多废话了）

教学开始

首先，你需要下载一个叫"termux"的手机应用。你可以在termux官方网站（termux.com）下载，或者在fdroid下载（推荐）。

下载好了吧？那么进入termux，先输入下面这段命令升级一下。（过程中需要点击回车）

pkg update && pkg upgrade  ##升级命令

升级完成后，授权termux手机的储存权限

termux-setup-storage  ##授权储存空间

接着下载今日主角，hugo！

pkg install hugo  ##安装hugo命令

下载完hugo后，你需要选择一个地方储存hugo的文件。这里我选择储存在手机sd卡内，不占用手机容量。

cd storage/shared  ##访问我的sd卡空间
hugo new site blog  ##创建名为"blog"的文件夹，并下载hugo文件到文件夹内

完成以上步骤后，hugo文件已经全部下载完成。我需要检查是否正常，于是我使用手机自带的文件管理器进入sd卡找到"blog"文件夹，下载好的文件如下。

接下来，返回termux输入命令进入"blog"文件夹

cd blog  ##访问文件夹

启动网页测试一下

hugo server  ##启动hugo

启动后前往浏览器，输入localhost:1313点击回车，会出现一片空白（毕竟还没创建博客的模板和内容，是正常的），但如果是出现"无法访问该网站，localhost 拒绝了我们的连接请求" 则是启动失败。 请重新检查你的步骤，文件夹是否正确等等。 一切正常后回到termux，点击CTRL后输入"C"关闭网页。

接下来，我们需要导入一个模板。打开浏览器，到https://themes.gohugo.io/选择一个你喜欢的模板，找到喜欢的模板的话点击download跳转到模板的github页面，这里我使用的是stack模板。

进入github页面后，点击绿色的"code"按钮，复制https网址后返回termux。输入

cd themes  ##进入hugo模板的文件夹
git clone https://github.com/CaiJimmy/hugo-theme-stack.git  ##此处的网址是刚刚github复制的模板的网址

完成后，“blog"文件夹里的"themes"文件夹，会有名为hugo-theme-stack的主题文件夹。

使用手机自带的文件管理器，进入blog  >  themes  >  hugo-theme-stack  >  exampleSite 找到"config.yaml"的文件。该文件里面都是一些网站的配置，可以按照喜好修改，关于配置的问题，我会在之后发文章解释。总之，复制"config.yaml"文件到"blog"的根目录内。删除原先的"config.toml”。

此时再次返回termux，运行网页。

cd ../  ##由于刚刚停留在theme文件夹，输入这段命令返回blog根目录
hugo server  ##启动网页

此时再次返回浏览器，刷新页面，你会发现你的网页已经成型了！
